    window 10 子系统 linux（centerOs 7） 安装redis

    安装步骤

    1. wget 下载最新的稳定版本redis（奇数版本号为开发版本，偶数版本为稳定版本， 推荐目录/usr/local/src/）

      wget http://download.redis.io/releases/redis-5.0.6.tar.gz

    2. 解压下载的安装包

      tar xzf redis-5.0.6.tar.gz

    3. redis 是基于c开发，需要安装c语言环境

      yum install gcc

    4. 检查gcc是否安装成功

      gcc –version

    5. 进入 redis-5.0.6 目录 编译安装

      cd redis-5.0.6
      make MALLOC=libc

    6. 将/usr/local/src/redis-5.0.6/src 目录下的文件加到/usr/local/bin目录

      cd src && make install
      以下提示添加成功
      Hint: It’s a good idea to run ‘make test’ ;)
      INSTALL install
      INSTALL install
      INSTALL install
      INSTALL install
      INSTALL install

    7. 修改redis-5.0.6/ 下redis.conf 文件 daemonize no 配置为 yes 后台启动
    8. 修改 /etc/rc.local 添加redis开机自启动

      vim /etc/rc.local
      redis-server /usr/local/src/redis-5.0.6/redis.conf （增加一行）

    补充 redis-server /usr/local/src/redis-5.0.6/redis.conf redis-server 以某某配置文件启动redis
